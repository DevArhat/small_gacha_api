# small_gacha_api

외환 유출 시뮬레이터
- - -
get /simulate/{game}/{target_pulls}

post /simulate/

post를 하면 request body에다가 파라미터를 넣어서 요청하는거라던데

어케하는거지?

- - -
target_pulls 파라미터는 돌파 기준

* 명함: 0
* 1돌: 1
* ...
- - -
/list 또는

/list/aliases 에서 구현된 확률모델 목록 확인가능

list의 id로 지정된 항목 뿐 아니라

aliases에 들어있는 영문 모두 사용할 수 있음

맞는 항목이 없으면 기본값은 붕괴스타레일

(원신이 기본값이었는데

별빛포착이 좀 복잡하기도 하고, 지금 시뮬레이터 확률이 호요버스 공지보다 1%정도 높게 나와서 바꿈
(100만번 기준 반천 픽업확률 약 56.2% 나옴, 호요버스 공지는 55%)

원신이랑 가장 확률설정이 비슷하면서 뽑기로직이 직관적이라서 붕스로 함)
